# MongoDB Half Stack

**Author**: Henrique Oliveira
**Version**: 1.0.0

## Overview

A vanilla nodejs http server utilizing mongodb to create a REST API. 

Includes standard `CRUD` methods:

*  `GET /<resource>`
* `POST /<resource>`
* `GET /<resource>/:id`
* `DELETE /<resource>/:id`
* `PUT /<resource>/:id`
